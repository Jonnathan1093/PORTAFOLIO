# 📊 Portafolio de Proyectos
---

## 👨‍💻 Acerca del Portafolio

Este repositorio reúne mis proyectos de **Análisis de Datos y Machine Learning**, desarrollados con herramientas como **Python, SQL, Power BI y Tableau**.  

Cada proyecto refleja un enfoque práctico: desde la **integración y limpieza de datos**, pasando por el **análisis exploratorio**, hasta la **visualización y modelado predictivo**, con el objetivo de **extraer valor de los datos y apoyar la toma de decisiones**.  

> ⚡ Los proyectos se actualizan periódicamente, incorporando nuevas técnicas, datasets y visualizaciones.
